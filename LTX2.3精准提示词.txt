## LTX 2.3 提示词三大模板（记录版）

---

### 模板一：文生视频（纯文本）

**使用场景**：无参考图，完全由文字描述生成视频。

**模板结构**：
```
整体氛围：[情绪、光影基调]
场景：[环境、地点、时间、细节]
主体+动作：[人物/物体描述，肢体动作，表情，与环境的交互]
镜头+焦距：[运镜方式，焦段，光圈，景深]
视觉风格：[画质、色调、质感、艺术风格]
运动/时间：[时长，帧率，快门，动态效果]
防护条件：无文字，无频闪，无logo
```

**示例**：
```
整体氛围：宁静雅致，午后柔光，高级感与亲和力并存。
场景：新中式高端住宅庭院，镂空雕花金属廊柱连廊，黑白大理石拼花地面镶嵌金色线条，日式枯山水与中式园林绿植环绕，远处现代高层住宅楼隐约可见，柔和午后自然光。
主体+动作：一位气质优雅的中年东亚女性，短发空气刘海，面带温和微笑，佩戴小巧珍珠耳钉，身穿墨绿色底花卉图案中式改良西装套装，双手交叠于腹前，轻微走动，面露自信。
镜头+焦距：缓慢推近运镜，从全身景别匀速推至胸部景别，50mm标准焦段，f/2.8光圈，焦点锁定人物面部。
视觉风格：8K超写实，商业人像摄影质感，色彩浓郁和谐，自然光影，细节精致。
运动/时间：总时长5秒，匀速推近，24fps，快门1/60秒，轻微动态模糊。
防护条件：无文字，无频闪，无logo。
```

### 模板二：图生视频（首帧+动态）

**使用场景**：有一张参考图作为首帧，提示词只描述动态变化和镜头运动，不重复画面细节（除非需要微调）。

**模板结构**：
```
[动态描述：人物动作、物体变化、镜头运动、时间节奏] + [防护条件]
（可选）画面微调：如需对参考图中某处做细微调整，可加一句，但不建议大改。
```

**要点**：
- 不写场景、主体外观（由参考图提供）
- 只写“变化”部分
- 帧率固定24fps
- 防护条件固定

**示例1（急速推近+速度变化）**：
```
第一秒内以急速非线性加速从远景快速推向社区大门中景，产生强烈速度感；第1秒结束时瞬间降速，剩余2秒转为极缓匀速推近，形成冲刺后滑行的节奏对比。24fps，前段快门1/250秒，后段1/60秒，动态模糊随速度变化自然过渡。无文字，无频闪，无logo。
```

**示例2（墨韵晕染微动）**：
```
哑光金色书法字体边缘的水墨飞白纹理，在3秒内以极缓速度向外晕染扩散，晕染范围不超过文字外轮廓的5%。晕染过程中，文字表面持续有极微弱的金色漂浮微粒，微粒随光束方向缓慢飘动。光束保持首帧位置不变。山峦轮廓与水波倒影绝对静止。24fps，快门1/60秒，无文字，无频闪，无logo。
```

### 模板三：图+音频驱动数字人视频（含面对镜头开心的讲解）

**使用场景**：有一张参考图（人物形象），有音频（口播），视频需生成人物动作和镜头运动，嘴型由音频自动驱动。  
**新增要点**：人物全程面对镜头，面带开心笑容，肢体配合讲解内容。

**模板结构**：
```
[身体动作 + 镜头运动 + 面对镜头 + 开心表情 + 时间节点（可选）] + [防护条件]
（严禁）任何关于“说话”“讲解”“嘴唇”“口型”的描述，避免干扰音频驱动。
```

**示例1（行走+面对镜头+开心）**：
```
人物向画面深处行走，摄影机缓慢跟随。行走过程中转头看向镜头，始终保持面对镜头，面露开心笑容，配合手势自然示意。全程动作从容优雅。24fps，无文字，无频闪，无logo。
```

**示例2（站立讲解+开心+微动）**：
```
人物居中站立，全程面对镜头，面露开心笑容，眼神明亮。双手自然配合音频内容做轻盈手势，身体有轻微前倾或侧转，始终保持与镜头的眼神交流。摄影机缓慢推近，从全身景别收至胸部景别。24fps，无文字，无频闪，无logo。
```

**示例3（含时间节点）**：
```
人物从画面左侧入画，向中心缓步行走，全程面对镜头方向。第3秒时停下，转身正对镜头，面露开心笑容，眼神直视镜头。第5秒开始抬手示意背景，手势自然。摄影机全程缓慢推近。24fps，无文字，无频闪，无logo。
```

---

**通用规则**：
- 帧率：24fps（固定）
- 防护条件：无文字，无频闪，无logo（固定）
- 音频驱动模式：提示词中禁止出现“说话”“讲解”“口型”等词
- 图生视频模式：首帧由参考图提供，提示词只写动态和镜头
- 文生视频模式：需完整描述场景、主体、动作、镜头、风格等
